Bubble Tones

Bubble tones is a customizable generative music program that produces pleasing, unpredictable, elevator-quality tone sequences.

Create additional bubbles by clicking or tapping. Use the controls to change the key, scale, and tempo.

This is an experiment by Chris Schnaars. Find the code and more info here. Reach out if you have technical feedback or ideas for improvements.
